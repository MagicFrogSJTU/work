## 排水量

| 术语         | 民用商船定义                            | 军舰定义                                       |
| ---------- | --------------------------------- | ------------------------------------------ |
| 空载排水量      | 无货物、燃油、淡水、备品、船员的空船重量              | 仅船体 + 固定设备，无燃油、弹药、补给、人员                    |
| 标准排水量      | 无此概念                              | 船体 + 武备 + 人员 + 全弹药，**不含**燃油、淡水、补给（军舰常用标注值） |
| 正常排水量      | 无此概念                              | 标准排水量 + 50% 燃油 / 淡水 / 补给，**军舰设计基准工况**      |
| 设计排水量      | 即**满载出港排水量**                      | 即**正常排水量**（设计核心工况）                         |
| 满载排水量      | 满载货物 + 足额燃油 / 淡水 / 备品，浮力 = 总重     | 标准排水量 + 100% 燃油 / 补给 + 全弹药（最大续航状态）         |
| 最大排水量      | 无专用定义                             | 超载至储备浮力极限的最大排水量（极少用）                       |
| 出港 / 到港排水量 | 出港：给养 100% 备足；到港：给养剩余 10%（用于续航核算） | 和民船一样                                      |
| 载重量（DWT）   | 满载排水量−空载排水量；**万吨轮即指此值**           | 无此概念（军舰不用载重量衡量）                            |
| 载货量        | 载重量−燃油 / 淡水 / 备品 / 船员等非货物重量       | 无此概念                                       |
| 总吨位（GT）    | 内部总容积折算的**容积吨位**（登记用）             | 无此概念                                       |
| 净吨位（NT）    | 载货容积折算，用于港口 / 运河费计费               | 无此概念                                       |
